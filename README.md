# Pet Clinic Patient Management System

## Overview

Pet Clinic Patient Management System is a full-stack web application developed to manage pet registrations, patient records, and user information efficiently.

## Features

- Pet Registration
- Patient Management
- User Registration and Login
- Profile Management
- Database Integration
- Responsive User Interface

## Technologies Used

### Frontend
- React.js
- JavaScript
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## Project Structure

### Frontend

- src/pages
- src/assets

### Backend
- models
- routes
- config.js
- index.js
